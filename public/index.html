Bienvenido a la Api de Book Shop

Podra recorrerla con los siguientes parametros

Parametros de busqueda:

  • Para ver la api tienes que agregarle a esta url la ruta/api
  • Para buscar los libros por su nombre la ruta es /api/libro/Titulo_del_Libro
  • Para buscar los libros por su isbn la ruta es /api/isbn/Numero_de_ISBN
  • Para buscar los libros que tienen cada autor la ruta es /api/autor/Nombre_del_Autor
  • Para buscar las editoriales que incluye la Api la ruta es /api/editorial
  • Para buscar los libros que tienen cada editorial la ruta es /api/editorial/Nombre_de_la_Editorial

Nota: Para el caso de que el parametro buscado tenga espacios puede buscarlo con los espacios(se pondra automaticamente %20 entre cada palabra) o unir las palabras con guion bajo (_).
